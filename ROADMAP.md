# Project Roadmap

## Phase 1
- Define vehicle search criteria
- Research approved listing and VIN data sources
- Build a basic command-line search prototype
- Normalize vehicle year, make, model, trim, mileage, price, and VIN fields

## Phase 2
- Add duplicate-listing detection
- Add market-comparison tools
- Add VIN decoding and recall lookup
- Add saved searches and notifications

## Phase 3
- Build a simple web interface
- Add user-configurable filters
- Publish installation and contribution documentation
- Invite outside contributors
