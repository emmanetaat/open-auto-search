# Open Auto Search

Open Auto Search is an open-source project designed to help used-car and auction buyers make better-informed purchasing decisions.

## Planned Features

- Search and organize publicly available used-car listings
- Match vehicles to user-defined criteria such as:
  - price
  - mileage
  - model year
  - trim
  - location
  - reliability
  - fuel economy
  - features
- Send alerts when strong matches appear
- Match auction VINs to publicly available:
  - vehicle specifications
  - recalls
  - prior listings
  - market comparisons
  - title and history clues
- Compare auction vehicles with similar retail listings

## Goal

Vehicle information is fragmented across listing sites, auction platforms, VIN databases, and manufacturer resources. This project aims to organize legally accessible public information into a free and understandable research tool.

## Status

Early planning and prototype stage.

## Responsible Data Use

The project will respect website terms, robots.txt rules, rate limits, privacy requirements, and applicable laws. It will prioritize approved APIs and publicly available data sources rather than bypassing access controls.
